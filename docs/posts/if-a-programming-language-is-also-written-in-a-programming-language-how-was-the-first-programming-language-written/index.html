第一个编写的编程语言没有写入，它们是构建的。数字计算机实际上是一种特定编程语言的电子实现，称为该计算机的机器语言。
要更容易拼写这一点，数字计算机是处理数字的设备，其存储在称为指令寄存器的地方，作为指令。每个数字都会导致计算机执行不同的动作。该操作由电线和电子组件实现。以下是一个非常原始计算机中可能的机器语言动作的一些示例。
000：什么都不做;加载下一个指令并继续。
001：将1添加到寄存器1中编码的数字。
002：从寄存器1中编码的数字中减去1。
003：如果寄存器1中编码的数字为0，则停止。
004：将寄存器1中编码的数字移动到指令寄存器中，并将其视为下一个指令。
等等。
该概念的重要方面是数字和动作之间的映射是完全任意的。
每种类型的计算机都有自己的机器语言，最早的程序是用这些机器语言编写的。他们非常难以阅读：典型的程序看起来像：
007 496 231 000 002 431 007 001 509 306 772 …
继续数百或数千个条目。在人们意识到的情况下，可以编写将更可读描述的程序进行翻译成机器语言的程序，并且开始开发更高级别语言的程序。
要返回基本点（也是在其他答案中的一些答案中进行的），数字计算机实际上是一种实现编程语言的机器。一旦你建造了一台电脑，你就不需要编写一个编程语言来开始，你有一个坐在你面前。
