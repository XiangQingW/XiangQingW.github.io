我开车过来，她被录取了。 SH被稳定，几天后有结果通过考试。预后不好。她唯一担心的是，三周后才能将其女儿22日生日。 （我的进入后，我只是几天，但是那是次要重要的……LOL）

在她到达时初始问题后，她从不需要疼痛缓解，从来没有真正的麻烦。她离开工作人员非常愚蠢，因为她完全了解了什么预期，但从未真正上下。 （她曾经是一名护士）
