回答 12

给予足够的时间与某人，我认为我会在开发之外寻找一些东西，作为他们作为开发人员的成功的指标。
当有人认为良好的开发人员时，它几乎总是关于技术能力，解决问题的能力等。但是，那些技能是一个良好开发人员的技能。
我会看看有人如何与他人互动。他们是否以非威胁的方式提供建设性的批评而不会居高临下？他们采取建议，让小事去吗？他们简明扼要地捍卫他们真正相信的东西吗？他们忠诚和谦卑他们的个人生活吗？
对我来说，这些是良好开发人员的指标。你可以成为那里的最聪明的人，但如果你试图向同龄人的背上移动到顶部并用精英主义的态度把别人放下，你就不会在今天的世界中伟大。
现在，当然存在技术组成部分，这有点难以预测。这是一种通用的声音，但如果一个人能够弄清楚的东西并超出问题和解决方案，那就是一个好兆头。
这有点好笑，但我期待很多（非常好的/高品质）的婚外，使良好的开发人员在技术意义上。刚刚介绍了一些问题，他们已经看到了解决方案，或者他们会查找现有的解决方案到类似的问题，并弄清楚如何应用它们。好吧，欢迎来到软件开发。
