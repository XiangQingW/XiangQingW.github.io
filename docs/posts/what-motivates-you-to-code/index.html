除了你可以通过建立东西来帮助自己来激励自己。我经常编写脚本来自动化很多东西，并简化我的编程任务。一些通常的任务包括在每个星期三早上自动下载Flash TV节目，或者自动检查我的个人Web服务器日志以获取错误，并通过通知向我报告给我。这项小型任务可以在提高整体生产力和动机方面进行很长的路要走。
另一个随机的黑客来激励自己是通过传奇编码人员阅读和聆听良好的演讲。当你倾听动机演讲时，突然爆发你得到的情绪可以令人难以理解。
我还使用粘滞便笺和计算成本/效益比，每个编程任务都是如此。这有助于让我优先考虑重要的东西，并激励我最大限度地提高影响指数。太多的粘滞便笺也迫使我完成长期的待定任务，并在墙上取出一份记录。
我的一位朋友使用非常酷的技巧。他买了啤酒。规则是他不能拥有它，除非他完成了日子任务。他有效地为自己创造了一个截止日期，让自己迫使自己在压力下工作。最后，他能够为他刚完成的任务奖励自己。

通常我面临着新的编程语言或技术，我对自己的ID说，喜欢学习。这个故事总是相同的。我看到了一些东西，我决定要做的事情，我下载每一块相关的软件和教程在互联网上，像疯狂一样疯狂，然后我一点点停下来，永远不会再回到它。
我真的学习的成功案例总是来自真正需要这样做的事情，因为我的工作取决于它。
我接受并与它的方式接受并和平。我知道这很难有足够的力量将能够为学习而努力学习。它总是更舒服，不要做它，即使你是非常动力的，你的动机也会最终消失，只是因为我们有了我们的起伏。
我对你的建议，如果我想真正学习一些东西，我会做什么，并确保我不会戒掉它是为了让一些重要的东西取决于它。在你的情况下，你说你把它作为一个项目经理工作。你为什么不从你自己管理的一个项目中分配一块东西？我不确切地知道管理项目的过程如何工作，但是让您说你的初步会议并为人们提供任务，以实际计划，你拍了一个简单的组成部分，你告诉其他人在这里照顾这篇文章，不要担心，我需要做一些编程工作，以便更多地参与整个项目。
